# My Dotfiles

This repository contains my personal dotfiles for configuring various applications and tools on my system.
These dotfiles are managed with stow and utilise multiple other tools.

## Useful Features

1. Keep your person git info in `~/.gitme` and this is loaded into the gitconfig
2. Use `switch <branch name, partial branch name or new (optional)>` to switch git branches easily
3. Use `<leader>P` in tmux to open a project finder
4. Auto WSL detection and configuration
